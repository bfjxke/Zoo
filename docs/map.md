## 连接关系图（拓扑结构）

```
                              ┌─────────┐
                              │ FOREST  │
                              └────┬────┘
                                   │
      ┌─────────┐            ╔═════╧═════╗            ┌─────────┐
      │ BASE_L  │────────────║  CENTER   ║────────────│ BASE_A  │
      └────┬────┘            ╚═════╤═════╝            └────┬────┘
           │                       │                         │
           │                       │                         │
           │                  ┌────┴────┐                     │
           │                  │ RIVER   │                    │
           │                  └────┬────┘                    │
           │                       │                         │
           │                  ┌────┴────┐                    │
           │                  │ BASE_N  │                    │
           │                  └─────────┘                    │
           │                                                │
           │                   ┌─────────┐                    │
           └───────────────────│MOUNTAIN │────────────────────┘
                               └─────────┘

说明：
• BASE_L ↔ CENTER ↔ BASE_A（直接连接）
• BASE_L ↔ FOREST（直接连接）
• BASE_A ↔ MOUNTAIN（直接连接）
• BASE_N ↔ RIVER（直接连接）
• 所有阵营基地都连接到CENTER
• 野外资源只连接自己的阵营基地和CENTER
```

## 移动路线示例

### 从守序基地到强势基地

```
步骤1: BASE_L → CENTER (消耗15耐力)
步骤2: CENTER → BASE_A (消耗15耐力)

总计: 30耐力，经过center暴露风险
```

### 从守序基地到森林（最近）

```
步骤1: BASE_L → FOREST (消耗15耐力)

总计: 15耐力，最快到达野外资源
```

### 从中立基地到山地（绕路）

```
步骤1: BASE_N → CENTER (消耗15耐力)
步骤2: CENTER → BASE_A (消耗15耐力)
步骤3: BASE_A → MOUNTAIN (消耗15耐力)

总计: 45耐力，需要经过多个节点
```

## 战略意义

| 区域 | 特点 | 战略价值 |
|------|------|----------|
| 中心节点(CENTER) | 连接所有节点 | 必争之地，暴露风险最高 |
| 阵营基地 | 复活点+私聊点 | 防守重地 |
| 野外资源 | 阵营专属+连接center | "后花园"，相对安全 |
